## Tutorial para rodar o projeto

- Instalar os pacotes do npm
```bash
npm install
```

- Iniciar servidor local
```bash
npm start
```

- Acessar via browser na url: http://localhost:5173/tech-farm/

## Versoes utilizadas

- Node: v18.15.0
- NPM: 9.5.0
